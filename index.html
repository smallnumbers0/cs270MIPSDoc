Comp Architecture

Computer architecture explains the functional behavior of computer systems. Most modern computers today go with the Von Neumann architecture (Remember this name because it will come up again later in RAM section). The Von Neumann architecture include a motherboard, CPU, RAM(main memory), secondary memory, and GPU, Input/Output Devices and many more. These hardware components of a computer all communicate together through something called a system bus. Imagine an actual bus picking up and dropping people off at different destinations. These busses hold important instructions that computers use to do computing

  • CPU

    The Central Processing Unit is where the computing occurs. The CPU includes registers, Arithmetic Logic Unit (ALU), and a control Unit(CU). If we want to tell the computer to add 1 + 2, many instructions and operations take place inside of the CPU in order to fulfill this request. For every instruction that occurs, the CPU must execute what we call a fetch-decode-execute cycle. Without going into abstract details of this cycle: During the fetch phase, the CPU fetches an instruction from main memory. Next, the decode phase determines the sequence of operations needed to be performed. And finally the execute phase is where that sequence of operations are performed.

  • RAM

    The RAM (Random Access Memory) is also known as the main memory. This piece of hardware temporarily keeps data accessible for your CPU so it does not need to find it from the secondary storage. RAM devices typically hold very little space ranging from 8GB to 32GB in most modern computers. Data is transffered between RAM and CPU through a system bus. This transfer process was great back then when CPU's were incredibly slow. A CPU's proccessing power today is incredibly fast and the data transfer using a system bus has proven to be a disadvantage due to the CPU needing to always idle and wait for information from RAM to process and carry out the instructions. This disadvantage is known as the Von Neumann bottleneck.

  • Secondary Storage

    Secondary Storage is our long term storage. Today most computers will use hard drives or solid state drives for this purpose. These devices can hold lots of data typically around 512GB - 1TB in modern personal computers.

  • I/O

    Input / Output devices include your keyboard, mouse, or other hardware that humans use to interact with the computer.

Decimal to Binary

The decimal number 65 in binary is 00100001. These ones and zeros are how a computer sees the number 65. Before we start converting a decimal number into binary, we first have to learn the powers of numbers. Decimal numbers are base 10 and binary is base 2.
